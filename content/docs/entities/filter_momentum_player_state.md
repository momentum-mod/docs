# filter_momentum_player_state

---

Filter that checks the player's movement state.

## Keyvalues

> **Player State** (player_state&lt;**choices**&gt;)

- **0**: "Grounded"
- **1**: "Surfing"
- **2**: "Bunnyhopping"

Default is Grounded; 0.
