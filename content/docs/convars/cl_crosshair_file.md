# cl_crosshair_file

Set the name of the custom VTF texture defined in `momentum/scripts/hud_textures.txt` to be used as a crosshair. Takes effect when [`{{ page.cvar_ref }}`](/var/{{ page.cvar_ref }}) is `1`.
