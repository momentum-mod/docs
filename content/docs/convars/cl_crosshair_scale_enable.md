# cl_crosshair_scale_enable

Toggles whether crosshair scales to the resolution defined in [`{{ page.cvar_ref1 }}`](/var/{{ page.cvar_ref1 }}). Takes effect when [`{{ page.cvar_ref2 }}`](/var/{{ page.cvar_ref2 }}) is `0`.
