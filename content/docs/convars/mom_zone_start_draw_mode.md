title: mom_zone_start_draw_mode

minimum_value: 0
maximum_value: 4
default_value: 1
---

Changes the draw mode of the start zone(s).
0 = Off; invisible, 1 = Outlines, 2 = Outlines as overlays (see through walls), 3 = Faces, 4 = Faces as overlays (see through walls).
